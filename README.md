## ✨ Функціонал

### 📋 Меню
- фільтрація по категоріях: **кава / чай / десерти**
- активні стани кнопок
- динамічне наповнення з JavaScript

### 🛒 Замовлення
- два типи:
  - **З собою**
  - **Забронювати столик**
- мультивибір товарів (кава + чай + десерт)
- автоматичний підрахунок суми
- валідація форми
- модальне вікно замість `alert`

### 🖼 Галерея
- адаптивна сітка
- **lightbox** з великим переглядом фото
- закриття по кліку та клавіші `Esc`

### 🍔 Навігація
- фіксований хедер
- **бургер-меню** для мобільних пристроїв
- плавна адаптація під різні екрани

### 🗺 Контакти
- вбудована Google Map
- адреса, телефон та години роботи
